HTML

  • Adds the structure and content of a webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Element refers to the entire code line while the tag refers to whats inside the code line
  • To link CSS and JavaScript to the HTML file we add a link element inside the headelement.

CSS

  • Adds how the webpage looks to users.
  • Inline CSS: styles an element directly in HTML file to a single line.
  • Internal CSS: is added to the header and effects all specified tags.
  • External CSS: (most common) keeps all CSS changes in a seperate file: easier to make changes. Link the file using placed in the head.
  • A margin indicates how much space we want around the outside of an element.
  • A padding includes how much space we want around the content inside an element.

GIT

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commit all files in the working branch.
  • git commit-m "Add Description": commit to repository with short description.

JavaScript

  • JavaScript allows for you to interact with the website.
  • Use a FOR loop.
  • The .js file extension idicates that it's going to be JavaScript.
  • Variable (var): a named container that allows us to store data in our code; name must be unique so we can referance it later. Ex: box labeled shape.
  • Assignment operator (=): to assign specific data we want stored in our variable. Ex: a circle in the box labeled shape.
  • Strings are enclosed in quotation marks.
  • Added under the footer element before the body element.
  • Control Flow: the order of which a computer executes a code. Using if statements.
  • Values can be truthy or falsy.
  • You can use else if for values that are neither truthy nor falsy.
  • Array: a single variable that holds group of similar data.
  • For a loop we need a starting point which is the first value of our array, then a condition, then the final statement telling it to loop back
